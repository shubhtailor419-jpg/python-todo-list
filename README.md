# Python To-Do List CLI

A simple command-line To-Do List application built with Python. This project allows users to manage their daily tasks by adding, viewing, and removing tasks through an interactive menu.

## Features

* Add new tasks
* View all existing tasks
* Remove completed or unwanted tasks
* User-friendly menu-driven interface
* Input validation for better user experience

## Technologies Used

* Python 3

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/your-username/python-todo-list.git
```

2. Navigate to the project folder:

```bash
cd python-todo-list
```

3. Run the program:

```bash
python todo.py
```

## Usage

When you run the program, you will see the following menu:

```text
===== TO-DO LIST =====
1. Add a task
2. View tasks
3. Remove a task
4. Exit
```

Choose an option by entering the corresponding number.

## Example

```text
===== TO-DO LIST =====
1. Add a task
2. View tasks
3. Remove a task
4. Exit

Enter your choice: 1
Enter a new task: Complete Python project
Task added.
```
